# 💰 Assistente Financeiro com IA Generativa

[![Streamlit App](https://img.shields.io/badge/Streamlit-FF4B4B?style=for-the-badge&logo=Streamlit&logoColor=white)](https://streamlit.io)
[![OpenAI](https://img.shields.io/badge/OpenAI-412991?style=for-the-badge&logo=openai&logoColor=white)](https://openai.com)

> Assistente virtual financeiro que combina IA generativa (OpenAI) com simuladores financeiros e uma interface moderna em Streamlit.

## ✨ Funcionalidades

- **💬 Chat Inteligente:** Tire dúvidas sobre finanças, investimentos, educação financeira.
- **📊 Calculadoras Financeiras:**
  - Juros compostos
  - Parcelamento Price e SAC
  - Meta de investimento (aporte mensal necessário)
- **📚 Conhecimento Financeiro:** Pergunte sobre qualquer conceito e receba explicações detalhadas.
- **📈 Comparador de Investimentos:** Compare o rendimento de diferentes ativos.
- **🖥️ Interface Profissional:** Design moderno, responsivo, com tabs e gráficos.

## 🛠️ Tecnologias

- Python 3.8+
- Streamlit – interface de usuário
- OpenAI GPT-3.5-turbo – modelo de linguagem
- Pandas, NumPy – manipulação de dados
- Matplotlib – visualização de gráficos
- python-dotenv – gerenciamento de variáveis de ambiente
